Fields

is_default_image_url

Indicates whether the icon image being returned is a default image, or is game-provided.

kind

Uniquely identifies the type of this resource. Value is always the fixed string games#eventDefinition.

display_name

The name to display for the event.

description

Description of what this event represents.

image_url

The base URL for the image that represents the event.

visibility

The visibility of event being tracked in this definition. Possible values are:
- "REVEALED": This event should be visible to all users. - "HIDDEN": This event should only be shown to users that have recorded this event at least once.

child_events

A list of events that are a child of this event.

id

The ID of the event.
